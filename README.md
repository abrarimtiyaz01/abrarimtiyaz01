## 👨‍💻 About Me

I am a B.Tech student in **Electronics and Communication Engineering at IUST**, passionate about blending low-level systems with high-level software. From developing robust Python automation scripts during my **internship at IIT Bombay** to training complex Deep Learning models for medical image segmentation, I love building secure, scalable, and intelligent systems.

- 🎓 **Education:** B.Tech in Electronics & Communication Engineering @ IUST (2022-Present).
- 💼 **Experience:** Summer Intern @ IIT Bombay (Linux systems, Bash scripting, Python automation).
- 🔭 **Currently working on:** Medical AI (U-Net architectures) and Advanced Image Segmentation.
- 🌱 **Focus Areas:** Deep Learning, Cybersecurity, Cloud Computing, and Network Architecture.
- ⚡ **Fun Fact:** I also design hardware! I've built physical circuits and power supplies using KiCad.
